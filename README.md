# Search Engine Tech - Demo
###Demonstrating:

1. Porter Stemmer
2. Language Detect
3. Part of Speech Tagging

## Install
[Download](http://nodejs.org/download/) and install Node.JS (comes with `npm`)

In main directory, install dependencies with `npm`
```shell
$ npm install
```

## Run
Run with `node` or `npm`
```shell
$ npm start
```
